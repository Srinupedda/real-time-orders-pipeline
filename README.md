# Real-Time Orders Data Streaming Pipeline (AWS)

This project demonstrates a real-time data streaming pipeline built using AWS services.

## Architecture
Python Data Generator → Amazon Kinesis Data Stream → AWS Lambda → Amazon S3 → Amazon Athena → Power BI Dashboard

## Technologies Used
- Python
- Amazon Kinesis
- AWS Lambda
- Amazon S3
- Amazon Athena
- Power BI

## Features
- Real-time order data generation
- Streaming data ingestion using Kinesis
- Serverless data processing with Lambda
- Raw data storage in S3
- SQL analytics using Athena
- Interactive dashboard visualization in Power BI

## Use Case
This pipeline simulates an e-commerce real-time analytics system to monitor:
- Total orders
- Revenue
- Product distribution
- City-wise sales
